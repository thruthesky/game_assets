# Game asset project

This project is for testing the animation and the player character monster NPC in 2.5D isometric. When the game asset content creator or monster player character creator works with this project, they first need to generate a 3D model from the trypo3d.ai website. Then they download the model and upload it to the Mixamo website for auto-rigging. Then they download the 3D model with auto-rig, and they open it in Blend. They attach the armature to the player character, then use this Flutter project to test the 3D model in 2.5D and with the animation also.
